Bieber as PDF

This simple demo shows how the Popcorn.js PDF plugin (based on Mozilla's pdf.js library) can be used. Popcorn's YouTube player is used to time the display of pages in an academic paper (PDF file) about Justin Bieber and Twitter. The Twitter plugin provides live tweets for the #bieber hash tag in order to provide context.

Popcorn's PDF plugin can be used to display PDF slides with audio or video from a talk, display background information for a documentary, or show any PDF file(s) at particular times within a media file without needing any plugins or additional software. Filmmakers with source materials in non-HTML formats can now produce PDF documents and have them rendered in Popcorn web experiences.
